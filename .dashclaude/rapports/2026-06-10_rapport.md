### Architecture VPN

```
SERVEUR (setup.ps1)                    CLIENT (Electron)
─────────────────                      ─────────────────
Installe Tailscale                     Demarre l'app
tailscale up → 100.64.x.x             Connecte LAN → SELECT _vpn_config
INSERT _vpn_config                     Stocke auth_key + tailscale_ip
                                       Si hors LAN → Tailscale embarque
                                       Tunnel SOCKS5 → 100.64.x.x:3306
```

### Pas d'installeur .exe

- setup.ps1 est un script PowerShell, pas un .exe
- L'app Electron peut etre buildee en .exe via `npm run electron:build`
- Les binaires Tailscale doivent etre telecharges et places dans tailscale-bin/
