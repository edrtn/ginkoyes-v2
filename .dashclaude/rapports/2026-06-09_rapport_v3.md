### 2. Photos produit CDN Sport2000

**Pattern URL universel** : `https://media.sport2000.fr/photos_plateforme_digitale/{MARQUE}/app-{REF_BASE}-{COU_CODE}-p.jpg`

**Logique `buildPhotoUrl`** :
- Extraction du premier mot de la marque DB (`PUMA ACC DOBOTEX` → `PUMA`)
- Strip du COU_CODE en suffixe de ART_REFMRK (séparateur espace ou tiret)
  - HOKA : `1162011 BWHT` → ref `1162011`, code `BWHT`
  - Converse : `M9166C-001` → ref `M9166C`, code `001`
  - Adidas : `IB7813 BLANC NOIR` → ref `IB7813`, code descriptif ignoré
  - Puma : `701225906 200` → ref `701225906`, code `200`
- **COU_CODE court (sans espace)** → inclus dans l'URL : `app-{ref}-{code}-p.jpg`
- **COU_CODE descriptif (avec espace)** → exclu : `app-{ref}-p.jpg`
- Fallback placeholder SVG si image 404 (`onError`)
