### Corrections encodage
- **Problème** : `\u00e9` en JSX text content affiché littéralement (pas interprété)
- **Cause** : unicode escapes ne fonctionnent qu'en JS string literals, pas en JSX text
- **Fix** : remplacé par `{"Références"}`, `{"Qté"}`, `{"€"}`, etc.

### Corrections image CDN

3. COU_CODE descriptif (`BLANC NOIR` chez Adidas) → ne pas inclure dans URL
   - Heuristique : espace dans COU_CODE = descriptif = pas dans URL
