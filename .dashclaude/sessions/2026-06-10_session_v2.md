## Debug iteratif sur PC reel

### Erreur 1 : $MyInvocation.MyCommand.Path null
- Cause : InnoSetup lance powershell.exe en mode hidden, $MyInvocation vide
- Fix : utiliser `$PSScriptRoot`, fallback sur `$InstallDir`

### Erreur 2 : Copy-Item self-copy
- Cause : `$srcDir -eq $InstallDir` quand lance depuis le dossier d'install
- Fix : skip copy si src=dest

### Erreur 3 : mysql stderr = erreur fatale
- Cause : `$ErrorActionPreference = "Stop"` + stderr mysql pipe
- Fix : try/catch avec `2>&1 | Out-Null`

### Erreur 4 : --production deprece
- Cause : Node.js v24 retire `--production`
- Fix : `--omit=dev`

### Erreur 5 : npm notices sur stderr
- Cause : npm ecrit notices/warnings sur stderr â†’ crash avec ErrorActionPreference=Stop
- Fix : temporairement `$ErrorActionPreference = "Continue"` pendant npm/tsc

### Erreur 6 : TypeScript introuvable
- Cause : typescript pas dans les deps du package.json genere
- Fix : ajout typescript, @types/node, @types/node-schedule

### Erreur 7 : BOM UTF-8 dans sync-config.json
- Cause : `Set-Content -Encoding UTF8` ajoute BOM (EF BB BF)
- Fix : `[IO.File]::WriteAllText($path, $content)` (UTF-8 sans BOM)

### Erreur 8 : Tables vides apres restauration
- Cause : SV.GBK est backup version 6 = Firebird 4.0, gbak 3.0 ne sait pas restaurer
- Fix : installer Firebird 4.0, mettre son chemin en priorite dans la detection

### Erreur 9 : node-firebird crash readUInt16LE
- Cause : node-firebird ne supporte pas le protocole SRP de Firebird 3+
- Fix : configurer Legacy_Auth + WireCrypt=Disabled dans firebird.conf, restart service

### Erreur 10 : SYSDBA auth failure
- Cause : mot de passe Firebird par defaut = 'masterkey', pas 'ginkoia'
- Note : sur Ginkoia reel le mot de passe est 'ginkoia'

### Erreur 11 : FK constraint sur TRUNCATE vide
- Fix : `SET FOREIGN_KEY_CHECKS=0` avant TRUNCATE dans le cas rows.length === 0

### Erreur 12 : gbak exit code 1 warnings
- Cause : gbak retourne 1 pour des warnings (table attribute not recognized)
- Fix : catch, verifier si FDB existe, continuer si oui

### Erreur 13 : gbak process bloque
- Fix : `taskkill /F /PID` pour liberer temp_sync.fdb
