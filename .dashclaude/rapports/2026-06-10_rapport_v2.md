### Lecons apprises
1. PowerShell `$MyInvocation.MyCommand.Path` est null quand lance via InnoSetup hidden
2. PowerShell `Set-Content -Encoding UTF8` ajoute un BOM, incompatible avec JSON.parse()
3. Firebird backup version 6 = Firebird 4.0 (pas 3.0), gbak 3.0 ne peut pas restaurer
4. node-firebird ne supporte pas SRP auth, necessite Legacy_Auth + WireCrypt=Disabled
5. gbak retourne exit code 1 pour des warnings non-fatals (table attributes)
6. Node.js v24 deprecie `--production`, remplace par `--omit=dev`
